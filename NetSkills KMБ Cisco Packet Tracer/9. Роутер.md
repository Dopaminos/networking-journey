Маршрутизатор - устройство, маршрутизирующее трафик между VLAN-ами. 
Нужно для того, чтобы лишний раз не покупать L3 коммутатор, если нагрузки низкие.

# Настройка вкратце

настраиваем access и trunk порты на l2 коммутаторах.
настраиваем транки с l3 коммутаторов на l2.
настраиваем роутер, добавляем на нём интерфейсы VLAN и назначаем им default gateway ip-шники.
__


Предположим, что каждый ПК находится в разных VLAN-ах. 
Настроим L2 switch:
```
en
conf t

vlan 2
vlan 3
vlan 4

in fa0/1
sw m a
sw a v 2
```

И так по каждому порту и нужным vlan-ам. У нас есть VLAN-ы выданные на каждый порт подключенный к компу. Теперь, нужно выдать транк на порт, подключенный к роутеру
```
in gi0/1
sw t e d
sw m t
sw t a v 2,3,4
```

___
не забыть `wr mem`
___
Все линки на роутере изначально выключены, поэтому тот, который подключен к L2 свичу надо поднять

```
in gi0/0
no shutdown
no ip address
```

Поскольку роутер работает с несколькими VLAN-ами по одному интерфейсу, для корректной работы с ними нужно создать subinterface для каждого:

```
in gi0/0.2
en d 2
ip ad 192.168.2.1 255.255.255.0
no shutdown
```

и так для 3 и 4 подсетей.

Роутер настроен, интерфейсы подняты! Настраиваем ПК и тестируем.
